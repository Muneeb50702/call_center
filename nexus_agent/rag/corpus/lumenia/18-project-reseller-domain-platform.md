## Lumenia Reseller Domain Platform Overview

Lumenia built the Reseller Domain Platform, a project in the Domain Registry / FinTech industry category. Lumenia's role on the project was Full-Stack Engineering.

Lumenia's project index summarizes it as a "Multi-hierarchy domain reseller system across registries."

The Reseller Domain Platform is the reference application behind Lumenia's Domain & Hosting Automation service line, whose page cites: "Multi-tier reseller portal with live registry sync and crypto wallet." Lumenia's AI Agentic Automation service page also cites "Seen in: AI domain search & reseller automation."

Lumenia does not publish a live URL, client name, timeline, or quantified results for the Reseller Domain Platform.

## Lumenia Reseller Domain Platform — What Lumenia Built

Lumenia lists five features it delivered for the Reseller Domain Platform, quoted verbatim:

1. "Multi-tier reseller hierarchy (Resellers/Sub-resellers)"
2. "Direct Verisign & Registry integrations"
3. "Intelligent domain search with description-based suggestions"
4. "Crypto wallet & Top-up system"
5. "Real-time domain availability tracking"

The "intelligent domain search with description-based suggestions" is the AI-assisted domain search capability Lumenia markets under its Domain & Hosting Automation service.

## Lumenia Reseller Domain Platform — Tech Stack

The technology stack Lumenia lists for the Reseller Domain Platform is:

- Laravel
- React
- Verisign API
- Cryptomus
- Blockonomics

The project index tags this project with Laravel and Registry API. This is one of only two projects in Lumenia's published portfolio that names React in its stack, the other being the EdTech LMS Platform.

## Lumenia Project — Master IDP Identity Provider

Lumenia built Master IDP (Identity Provider), a project in the Identity Management / SSO industry category, published at https://lumenialab.com/projects/master-idp. Lumenia's role was Full-Stack Engineering.

Lumenia's project index summarizes Master IDP as a "Multi-tenant SSO and identity management system."

What Lumenia built for Master IDP, quoted verbatim:

1. "Centralized SSO for WHMCS, CRM, and Portals"
2. "Multi-tenant organization management"
3. "Unified wallet for cross-app transactions"
4. "Crypto integration (Blockonomics & Cryptomus)"
5. "Tiered subscription and plan management"

Tech stack Lumenia lists for Master IDP: Laravel, PHP, MySQL, SSO (OpenID Connect), Wallet Services, and Blockonomics (Crypto). Tags: Laravel, SSO, Multi-tenant, FinTech, Crypto.

Lumenia does not publish a live URL, client name, or results for Master IDP.

## Lumenia Domain, Hosting and Crypto Experience Summary

Across the Reseller Domain Platform and Master IDP, Lumenia's published domain, hosting and crypto experience includes: Verisign and registry API integration, multi-tier reseller and sub-reseller hierarchies, real-time domain availability tracking, AI/description-based domain search, centralized SSO via OpenID Connect spanning WHMCS, CRM and portals, unified cross-app wallets, tiered subscription management, and crypto payment integration via Blockonomics and Cryptomus.

Lumenia's Domain & Hosting Automation service page additionally markets Verisign / CentralNic / PIR APIs, WHMCS replica and plugins, AI domain search, and Plesk / cPanel automation. Note that CentralNic APIs, PIR APIs, WHMCS replica work, and Plesk/cPanel automation are marketed as service capabilities but are not named on any individual Lumenia project page.
