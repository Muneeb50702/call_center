## Lumenia Medaan Project Overview

Lumenia built Medaan, a project in the FoodTech / E-commerce industry category. Lumenia's role on the project was Full-Stack Engineering. The live URL Lumenia publishes for Medaan is https://medaan.ca

Lumenia describes Medaan verbatim on its homepage as: "Architected and engineered a comprehensive full-stack food delivery and brand experience platform tailored for the Canadian market. Developed a secure, session-based authentication system utilizing Laravel Breeze. The platform features an immersive frontend with a dynamic hero section and a sophisticated custom admin portal for managing categories, dishes, deals, and discounts."

Lumenia's project index summarizes Medaan as a "Food delivery and brand experience platform for Canada."

Lumenia does not publish a client name, timeline, or quantified results for Medaan.

## Lumenia Medaan — What Lumenia Built

Lumenia lists four things it built for Medaan:

1. Secure authentication via Laravel Breeze
2. Custom admin dashboard
3. Deliverect API integration
4. Dynamic promotional engine

The Medaan admin portal Lumenia built manages categories, dishes, deals, and discounts. The frontend Lumenia built features a dynamic hero section as part of the brand experience.

## Lumenia Medaan — Tech Stack

The technology stack Lumenia lists for Medaan is:

- Alpine.js
- Deliverect API

The project index tags Medaan with Laravel and Tailwind CSS.
