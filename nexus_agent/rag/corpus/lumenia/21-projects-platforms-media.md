## Lumenia Project — Peer-to-Peer Rideshare Platform

Lumenia built a Peer-to-Peer Rideshare Platform, a project in the Ride-Sharing / Mobility-as-a-Service category, published at https://lumenialab.com/projects/rideshare-platform. Lumenia's role was Full-Stack Engineering.

Lumenia's project index summarizes it as a "Real-time ride-matching system with live tracking."

Key features Lumenia built:

1. Real-time peer-to-peer ride matching engine
2. Live geolocation tracking and location sharing
3. Real-time in-app chat infrastructure
4. Dynamic fare and fleet management

Tech stack Lumenia lists: full-stack web technologies, real-time WebSockets, Geolocation APIs. Tags: WebSockets, Geolocation, Ride-Sharing, Real-time.

The Lumenia project page shows both desktop and mobile interface mockups. Lumenia does not publish a live URL, client name, or quantified results for this project. Lumenia's SaaS solution page separately names a "911 Limo App" described as ride-sharing with live tracking, which has no dedicated project page.
